# Standard Stream Objects

- cin (stdin)

- cerr (stderr)

# File Access

- ifstream (input file)
- ofstream (output file)

## `<<` Insertion Operator

Inserts sequence of characters into a stream buffer.

## `>>` Extraction Operator

Extracts data from the string buffer into variables. Stops at first whitespace.
